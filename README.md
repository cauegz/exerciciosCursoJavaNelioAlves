# ☕ Java Completo - Programação Orientada a Objetos
### Curso do Prof. Nelio Alves (Udemy)

Este repositório contém as soluções dos exercícios e projetos desenvolvidos por mim durante o curso **Java Completo Programação Orientada a Objetos + Projetos**, ministrado pelo Prof. Nelio Alves. O foco principal é o domínio da linguagem Java seguindo as melhores práticas de mercado e o paradigma de POO.

---

## 🚀 Tecnologias e Ferramentas
* **Linguagem:** Java 21+ (LTS)
* **IDE:** IntelliJ IDEA
* **Paradigma:** Programação Orientada a Objetos (POO)

---

## 📌 Principais Conceitos Aplicados

Durante o desenvolvimento dos exercícios, foram aplicados conceitos fundamentais para a construção de softwares robustos:

* **Abstração e Encapsulamento:** Criação de classes com atributos privados e métodos de acesso.
* **Composição de Objetos:** Implementação de relações "Tem-um", onde um objeto contém instâncias de outros.
* **Enumerações (Enums):** Uso de tipos enumerados para constantes como níveis de cargo e status de pedidos.
* **Manipulação de Listas:** Uso de `ArrayList` para gerenciar coleções dinâmicas de dados.
* **Tratamento de Exceções:** Controle de erros de entrada de dados e parsing de datas.

---

## 📂 Destaque: Exercício de Composição (Worker & Contracts)
Um dos módulos mais importantes documentados aqui é o de **Composição**, onde foi desenvolvido um sistema para calcular a renda de um trabalhador com base em múltiplos contratos por hora.

**Desafios superados:**
- Gerenciamento de associações entre classes.
- Lógica de filtragem de listas por mês e ano.
- Formatação e parsing de datas com `DateTimeFormatter`.
